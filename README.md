**Cotizador Criptomonedas** es una aplicacion web para la cotizacion del precio en tiempo real de distintas criptomonedas. El proyecto está desarrollado utilizando las siguientes tecnologías:

* HTML
* JavaScript
* React
* Vite
* API cryptocompare
